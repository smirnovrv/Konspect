## Назначение конфигурации

Конфигурация предназначена для ведения и структурирования конспектов: хранения заметок, тем, лекций, тегов и связанных файлов.
Состав объектов указывает на поддержку форматированного текста/HTML-структур и привязки медиа-данных к заметкам.
